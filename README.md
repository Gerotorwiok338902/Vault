# 🔐 VAULT — Encrypted Private Messenger

Prywatny komunikator z szyfrowaniem AES-256-GCM działający bezpośrednio w przeglądarce.

## Demo

Dostępny po włączeniu GitHub Pages pod adresem:
`https://[twoja-nazwa].github.io/vault/`

## Funkcje

- **AES-256-GCM** — każda wiadomość szyfrowana lokalnie w przeglądarce
- **Zero-knowledge** — serwer nigdy nie widzi treści wiadomości ani hasła
- **Ochrona ekranu** — przy zrzucie ekranu / przełączeniu okna → czarny ekran
- **Wiele urządzeń** — działa na telefonie, tablecie, komputerze
- **Bez rejestracji** — wystarczy link + nazwa kanału + hasło

## Jak używać

1. Otwórz link na obu urządzeniach
2. Wpisz **tę samą nazwę kanału** i **to samo hasło** (min. 10 znaków)
3. Podaj różne pseudonimy
4. Rozmawiaj — wiadomości synchronizują się w czasie rzeczywistym

Dostęp do kanału mają tylko osoby znające nazwę kanału **i** hasło.

## Instalacja na GitHub Pages

1. Stwórz nowe repozytorium na GitHub (np. `vault`)
2. Wgraj plik `index.html` do repozytorium
3. Wejdź w **Settings → Pages → Source → main branch**
4. Poczekaj chwilę — strona będzie dostępna pod adresem `https://[twoja-nazwa].github.io/vault/`
